Library Management System (School Project)

This is a simple C++ console-based Library Management System project that uses the Qt library. 
Through a console/terminal menu, this application will allow you to add, display, borrow, and return books 
and magazines. Data is stored in a local file so your library will persist between sessions.  

---  

** Features  

- Add new books and magazines to your collection/library.  

- Display all library items and their borrowed statuses.  

- Borrow and return books and magazines by their ID.  

- User-friendly and menu-driven interface.  

- Data is stored in `library_data.txt`.  

---  

** Getting Started  

** Prerequisites  

- [Qt 5 or Qt 6](https://www.qt.io/download) 

- [CMake] version 3.16 or newer  

- A C++17 compatible compiler 

---  

** Building with Qt Creator (Recommended)  

1. Open Qt Creator.

2. Select `File > Open Files or Project...` and also select `CMakeLists.txt` from this project.  

3. Configure your project (in the main.cpp, under the main program, type in the correct path/dr where you placed 
your library_data.txt file). 

4. Build the project (`Build` button or `Ctrl+B`).  

5. Run the app with the `Run` button.

---

** How to Use.

When you run the program, you will see a numbered menu. 
All you need to do is enter a number/option that you want and tap enter on your keyboard.
